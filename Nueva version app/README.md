# Documentación de la aplicación

Esta es una breve guía sobre la aplicación Shiny que has desarrollado
para consultar y visualizar datos de una base de datos.

## Descripción

La aplicación Shiny proporciona una interfaz de usuario interactiva para
realizar consultas a una base de datos y visualizar los resultados en
forma de gráficos y tablas.
